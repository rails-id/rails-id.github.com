Menginstal Ruby dan Rails di macOS

Menginstal Homebrew

Jalankan perintah berikut untuk menginstall Homebrew: {% highlight bash session %} /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" {% endhighlight %}

Menginstal RVM

Menginstal GPG keys

Sebagai langkah awal, menginstal GPG keys untuk memverifikasi paket yang diinstal: {% highlight bash session %} gpg --keyserver hkp://pool.sks-keyservers.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB {% endhighlight %}

Menginstal RVM versi stabil terbaru

{% highlight bash %} \curl -sSL https://get.rvm.io | bash -s stable {% endhighlight %}

Cek versi RVM

Anda dapat mengecek versi RVM terinstal dengan menjalankan perintah berikut: {% highlight bash %} rvm -v {% endhighlight %}

Menginstall Ruby

Jalankan perintah berikut untuk menginstall Ruby versi 3.0.0 {% highlight bash %} rvm install 3.0.0 {% endhighlight %}

Cek versi ruby

Anda dapat mengecek versi RVM terinstal dengan menjalankan perintah berikut: {% highlight bash %} ruby -v {% endhighlight %}

Menginstal Ruby on Rails

Jalankan perintah berikut untuk menginstall Ruby on Rails versi 6.1.1 {% highlight bash %} gem install rails -v 6.1.1 {% endhighlight %}

Cek versi Ruby on Rails

Anda dapat mengecek versi Ruby on Rails terinstal dengan menjalankan perintah berikut: {% highlight bash %} rails -v {% endhighlight %}

Menginstal Database

Langkah selanjutnya yakni menginstal database, Anda dapat memilih salah satu dari database berikut:

SQLite

Jalankan perintah berikut untuk menginstal SQLite: {% highlight bash %} brew install sqlite3 {% endhighlight %}

MySQL

Jalankan perintah berikut untuk menginstal MySQL: {% highlight bash %} brew install mysql {% endhighlight %}

Menjalankan service MySQL

{% highlight bash %} brew service start mysql {% endhighlight %}

PostgreSQL

Jalankan perintah berikut untuk menginstal PostgreSQL: {% highlight bash %} brew install postgresql {% endhighlight %}

Menjalankan service PostgreSQL

{% highlight bash %} brew service start postgresql {% endhighlight %}

Mulai membangun aplikasi dengan Ruby on Rails

Pilih salah satu perintah berikut untuk membangun aplikasi pertamamu dengan Ruby on Rails menggunakan database pilihanmu {% highlight bash %} # untuk membuat aplikasi menggunakan sqlite3 rails new nama_aplikasimu # untuk membuat aplikasi menggunakan sqlite3 rails new nama_aplikasimu -d mysql # untuk membuat aplikasi menggunakan sqlite3 rails new nama_aplikasimu -d postgresql {% endhighlight %} Langkah selanjutnya, masuklah ke folder nama_aplikasimu dengan cara: {% highlight bash %} cd nama_aplikasimu {% endhighlight %} Membuat database untuk aplikasimu {% highlight bash %} rails db:create {% endhighlight %} Menjalankan aplikasimu: {% highlight bash %} rails server # atau rails s {% endhighlight %}
